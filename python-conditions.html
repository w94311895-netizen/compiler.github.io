Python If-Else Conditions: A Complete Beginner's Guide

Master conditional statements and decision making in Python. Learn if, elif, and else with practical examples.

What Are Conditionals?

Conditionals let programs make decisions. "If it's raining, take an umbrella. Otherwise, don't." This is conditional logic.

Without conditionals, programs would execute the same steps every time, regardless of the situation. Conditionals make programs intelligent and responsive.

The if Statement

The basic if statement executes code only if a condition is true:

age = 18
if age >= 18:
print("You are an adult")

If the condition is true, the indented code runs. If false, it's skipped.

Comparison Operators

Comparison operators test relationships between values:

  • == Equal to
  • != Not equal to
  • < Less than
  • > Greater than
  • <= Less than or equal to
  • >= Greater than or equal to
x = 10
print(x == 10) # True
print(x != 5) # True
print(x < 15) # True

if-else Statements

The else clause executes when the condition is false:

age = 15
if age >= 18:
print("You are an adult")
else:
print("You are a minor")

if-elif-else Statements

Check multiple conditions with elif (else if):

score = 85
if score >= 90:
print("Grade: A")
elif score >= 80:
print("Grade: B")
elif score >= 70:
print("Grade: C")
else:
print("Grade: F")

Python checks each condition in order. The first true condition executes, then the rest are skipped.

Logical Operators

and Operator

Both conditions must be true:

age = 25
has_license = True
if age >= 18 and has_license:
print("You can drive")

or Operator

At least one condition must be true:

day = "Saturday"
if day == "Saturday" or day == "Sunday":
print("It's the weekend!")

not Operator

Reverses a boolean:

is_raining = False
if not is_raining:
print("No rain today!")

Practical Examples

Example 1: Voting Eligibility

age = 17
if age >= 18:
print("You can vote")
else:
print("You must be 18 to vote")

Example 2: Password Validation

password = "secret123"
correct_password = "secret123"
if password == correct_password:
print("Access granted")
else:
print("Access denied")

Deep Practice: Writing Real Decision Rules

Conditionals turn a script into a program that reacts to different situations. For example, a grading program can decide whether a student passed, needs review, or earned an excellent score. The key is to write conditions in an order that makes sense.

score = 86

if score >= 90:
    print("Excellent")
elif score >= 60:
    print("Passed")
else:
    print("Needs more practice")

Common mistakes

  • Using = instead of == for comparison.
  • Putting broad conditions before specific conditions.
  • Forgetting indentation under if, elif, or else.

Practice exercises

  1. Write a program that checks if a number is positive, negative, or zero.
  2. Create a discount rule based on a customer's purchase total.
  3. Check if a password is long enough and contains no spaces.

FAQ

Can I have more than one elif? Yes. Use as many as needed, but keep the order clear.

Does else need a condition? No. It runs when previous conditions are false.

Conclusion

Conditions are best learned by testing both true and false cases. Write small rules first, then combine them into larger programs.

Condition practice checklist

Practice conditions by checking grades, login status, temperature, and discount rules. Write the simplest condition first, then add elif branches only when the program really has more than two outcomes.

  • Use comparison operators carefully.
  • Keep indentation consistent.
  • Test true and false cases so every branch is checked.
