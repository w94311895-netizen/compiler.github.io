Python Data Types: Complete Guide

Last updated: June 18, 2026 | Reading time: 8 minutes

Introduction to Python Data Types

Python provides several built-in data types for storing and manipulating data. Understanding these types is fundamental to writing effective Python code. Each type has specific characteristics and methods.

1. Numeric Types

Integers (int)

Integers are whole numbers without decimal points. Python supports arbitrarily large integers.

age = 25
temperature = -10
big_number = 123456789012345678901234567890
print(type(age))  # <class 'int'>
print(age + 5)    # 30

Floats (float)

Floats represent decimal numbers. Use them for precise calculations involving decimals.

price = 19.99
percentage = 0.75
scientific = 1.5e-3  # Scientific notation
print(type(price))    # <class 'float'>
print(price * 2)      # 39.98

Complex Numbers

Python natively supports complex numbers for mathematical operations.

z = 3 + 4j
print(z.real)      # 3.0
print(z.imag)      # 4.0
print(z.conjugate())  # (3-4j)

2. String Type (str)

Strings are sequences of characters used for text data. They're immutable and support many operations.

name = "Python"
greeting = 'Hello, World!'
multiline = """This is
a multiline
string"""

# String operations
print(name.lower())        # python
print(name.upper())        # PYTHON
print(name[0])             # P
print(len(name))           # 6
print(name * 2)            # PythonPython
print("Py" in name)        # True

3. Sequence Types

Lists (list)

Lists are ordered, mutable collections that can store multiple data types.

fruits = ["apple", "banana", "cherry"]
mixed = [1, "hello", 3.14, True]

# List operations
fruits.append("date")           # Add item
fruits.remove("banana")         # Remove item
print(fruits[0])                # Access first item
print(fruits[-1])               # Access last item
print(len(fruits))              # Get length
print(fruits[:2])               # Slice first 2 items

Tuples (tuple)

Tuples are ordered, immutable collections. Use them when data shouldn't change.

coordinates = (10, 20)
rgb = (255, 128, 0)

print(coordinates[0])     # 10
print(len(coordinates))   # 2
a, b = coordinates        # Unpacking
print(a, b)               # 10 20

4. Set Types

Sets (set)

Sets are unordered collections of unique items. Great for removing duplicates.

colors = {"red", "green", "blue"}
numbers = {1, 2, 2, 3, 3, 3}  # Duplicates removed

print(numbers)                    # {1, 2, 3}
colors.add("yellow")             # Add item
colors.remove("red")             # Remove item
print("blue" in colors)           # True

5. Dictionaries

Dictionaries store key-value pairs and are highly efficient for data lookup.

person = {
    "name": "Alice",
    "age": 30,
    "city": "New York"
}

print(person["name"])         # Alice
print(person.get("age"))      # 30
person["email"] = "alice@example.com"  # Add item
