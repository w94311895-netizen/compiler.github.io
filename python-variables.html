Python Variables Tutorial

Updated: June 14, 2026 | Reading time: 5 minutes

What Are Variables?

Variables are containers for storing data values. In Python, a variable is created the moment you assign a value to it. You don't need to declare the type - Python figures it out automatically.

Creating Variables

To create a variable, you use the assignment operator (=). Here's the syntax:

variable_name = value

Example:

name = "John"
age = 25
height = 5.9
is_student = True

Variable Naming Rules

Follow these rules when naming variables:

  • Must start with a letter (a-z, A-Z) or underscore (_)
  • Cannot start with a number
  • Can only contain alphanumeric characters and underscores (A-z, 0-9, and _)
  • Variable names are case-sensitive (age, Age, and AGE are different)
  • Use descriptive names that explain what the variable stores

Valid and Invalid Variable Names

✓ Valid:
my_name = "Alice"
user_age = 30
_private_var = 100
CONSTANT_VALUE = 3.14

✗ Invalid:
123abc = 50  (starts with number)
my-var = 10  (contains hyphen)
my var = 20  (contains space)
class = 5    (reserved keyword)

Data Types

Python has several built-in data types:

  • str: Text data - "Hello World"
  • int: Integer numbers - 42
  • float: Decimal numbers - 3.14
  • bool: Boolean values - True or False
  • list: Ordered collection - [1, 2, 3]
  • dict: Key-value pairs - {"name": "John"}

Type Conversion

Convert between data types using built-in functions:

x = 42           # integer
y = str(x)       # convert to string: "42"
z = float(x)     # convert to float: 42.0
a = int("100")   # convert to integer: 100

Practical Example

name = "Alice"
age = 25
gpa = 3.85
is_graduated = False

print(f"Name: {name}")
print(f"Age: {age}")
print(f"GPA: {gpa}")
print(f"Graduated: {is_graduated}")

Deep Practice: Building a Student Profile with Variables

A useful way to understand variables is to build a small student profile. Each value has a purpose: text stores names, numbers store scores, and booleans store yes-or-no states. When you choose clear variable names, the program becomes easier to read and easier to debug later.

student_name = "Ava"
student_age = 16
math_score = 88
english_score = 91
is_enrolled = True

average_score = (math_score + english_score) / 2
print(f"{student_name} average: {average_score}")

Common mistakes

  • Using names like x and y when the value has a real meaning.
  • Putting numbers inside quotes and then wondering why math does not work.
  • Changing capitalization, such as using Score in one place and score in another.

Practice exercises

  1. Create variables for a book title, author, page count, and whether you finished reading it.
  2. Calculate the total cost of three products using separate price variables.
  3. Print a friendly sentence using an f-string and at least three variables.

FAQ

Do I need to declare a variable type in Python? No. Python detects the type from the value you assign.

Can a variable value change? Yes. Reassigning a variable gives it a new value.
