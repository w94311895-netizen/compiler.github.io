# YBY Compiler - Free Online Multi-Language Code Editor

YBY Compiler is a comprehensive, browser-based coding platform designed for students, beginners, and educators. It enables you to write, run, and test code in **Python, C, C++, and Java** directly from your web browser—with no installation, no setup, and completely free.

## Features

### Core Capabilities
- **Python Execution**: Runs Python directly in your browser using Pyodide technology
- **Multi-Language Support**: C, C++, and Java via secure cloud API
- **Real-Time Output**: Instant code execution with live console output
- **Error Explanations**: Technical errors translated to beginner-friendly language
- **Chart Visualization**: Built-in support for Matplotlib and Seaborn plots
- **File Uploads**: Attach CSV, JSON, TXT, TSV, XLSX, and Parquet files to Python scripts
- **Code Improvements**: Automatic suggestions for cleaner, safer code

### Supported Libraries & Tools

**Python:**
- NumPy - Numerical computing and array operations
- pandas - Data manipulation and analysis
- Matplotlib - Data visualization and charting
- seaborn - Statistical graphics
- scikit-learn - Machine learning examples
- Standard library modules

**Other Languages:**
- C: Standard library, console programs
- C++: STL, console applications
- Java: Core Java libraries, console output

## Getting Started

### 1. Write Code
Select your programming language (Python, C, C++, or Java) and type or paste your code into the editor.

### 2. Upload Datasets (Python Only)
Click "Upload dataset(s)" to attach CSV, JSON, or other supported files. Access them in your code at:
```python
df = pd.read_csv("/home/pyodide/data/filename.csv")
```

### 3. Run Your Code
Click the "Run Code" button to execute instantly. View:
- Console output
- Error messages with explanations
- Code improvement suggestions
- Chart visualizations (for Matplotlib/Seaborn)

## Technical Architecture

- **Frontend**: Static HTML/CSS/JavaScript (no backend dependencies for Python)
- **Python Runtime**: Pyodide (WebAssembly-based CPython)
- **C/C++/Java Execution**: Secure cloud compilation API
- **Hosting**: Netlify CDN for fast, reliable access
- **Security**: No code storage, no login required, privacy-first design
