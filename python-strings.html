📝 Python Strings

Strings are sequences of characters used to store text data. Learn how to create, manipulate, and format strings like a professional.

What is a String?

A string is a sequence of characters enclosed in quotes. Strings in Python are:

  • Immutable: Cannot be changed after creation (but can be reassigned)
  • Ordered: Each character has a position (index)
  • Flexible: Can contain letters, numbers, symbols, and spaces
  • Indexable: Access characters using their position

Creating Strings


# Single quotes
name = 'Alice'

# Double quotes
message = "Hello, World!"

# Triple quotes (multiline)
poem = '''Roses are red,
Violets are blue,
Python is fun,
And coding is too!'''

# Using escape characters
quote = "She said, \"Python is awesome!\""
newline = "Line 1\nLine 2"
tab = "Column1\tColumn2"

# Empty string
empty = ""

Accessing String Characters

Using Index (Positive)


text = "Python"

print(text[0])    # P
print(text[1])    # y
print(text[5])    # n

Using Negative Index


text = "Python"

print(text[-1])   # n (last character)
print(text[-2])   # o (second from last)
print(text[-6])   # P (first character)

String Slicing


text = "Python Programming"

print(text[0:6])     # Python
print(text[7:])      # Programming
print(text[:6])      # Python
print(text[::2])     # Pto rgamn
print(text[::-1])    # gnimmargorP nohtyP (reversed)

String Methods

Case Methods


text = "Hello World"

print(text.upper())        # HELLO WORLD
print(text.lower())        # hello world
print(text.capitalize())   # Hello world
print(text.title())        # Hello World
print(text.swapcase())     # hELLO wORLD

Search and Replace


text = "Python is great. Python is fun."

# Find position
pos = text.find("is")                    # 7
print(pos if pos != -1 else "Not found")

# Count occurrences
count = text.count("Python")             # 2

# Replace
new_text = text.replace("Python", "Java")
print(new_text)  # Java is great. Java is fun.

# Start and end check
print(text.startswith("Python"))        # True
print(text.endswith("fun."))            # True

Trimming Whitespace


text = "   Hello, World!   "

print(text.strip())      # Hello, World!
print(text.lstrip())     # Hello, World!   (left trim)
print(text.rstrip())     #    Hello, World! (right trim)

Splitting and Joining


# Split string into list
sentence = "Python is fun and easy"
words = sentence.split()  # ['Python', 'is', 'fun', 'and', 'easy']

# Split with separator
csv = "apple,banana,orange"
fruits = csv.split(",")   # ['apple', 'banana', 'orange']

# Join list into string
words = ["Python", "is", "awesome"]
sentence = " ".join(words)  # "Python is awesome"

String Formatting

Method 1: Using % (Old Style)


name = "Alice"
age = 25
print("Name: %s, Age: %d" % (name, age))
# Name: Alice, Age: 25

Method 2: Using .format()


name = "Bob"
age = 30
gpa = 3.85
print("Name: {}, Age: {}, GPA: {}".format(name, age, gpa))
print("Name: {0}, Age: {1}".format(name, age))
print("Age: {age}, Name: {name}".format(name=name, age=age))

Method 3: Using f-strings (Best)


name = "Charlie"
age = 35
gpa = 3.92
print(f"Name: {name}, Age: {age}")
print(f"GPA: {gpa:.2f}")  # 3.92 (2 decimal places)
print(f"{name} is {age} years old")
print(f"Next year: {age + 1}")

String Checks


text = "Hello123"

print(text.isalpha())           # False (has numbers)
print(text.isdigit())           # False (has letters)
print(text.isalnum())           # True (only letters and numbers)
print(text.isspace())           # False
print(text.isupper())           # False
print(text.islower())           # False

# Custom checks
print("123".isdigit())          # True
print("abc".isalpha())          # True

String Concatenation


# Method 1: Using +
first_name = "John"
last_name = "Doe"
full_name = first_name + " " + last_name

# Method 2: Using f-strings
full_name = f"{first_name} {last_name}"

# Method 3: Using join()
words = ["Hello", "World"]
message = " ".join(words)

# Method 4: Using format()
message = "{} {}".format(first_name, last_name)
